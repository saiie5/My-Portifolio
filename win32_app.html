Win32 App Deployment in Intune

1. Prepare the Win32 App

Convert the app to the Intune format using the Microsoft Win32 Content Prep Tool.

Steps:

  • Download the Win32 Content Prep Tool.
  • Run the tool to package the `.exe` or `.msi` file into `.intunewin` format.

2. Upload and Configure in Intune

Use the Intune portal to deploy the packaged application.

Steps:

  • Go to **Microsoft Endpoint Manager** → **Apps** → **Windows** → **Add**.
  • Choose **Win32 app**, upload the `.intunewin` file, and configure requirements.
  • Set up dependencies, detection rules, and assignments.

3. Monitor Deployment

Check Intune logs and reports to verify deployment success.

Steps:

  • Navigate to **Monitor** in the Intune portal.
  • View device installation status and troubleshoot failed deployments.

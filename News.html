Intune Weekly Update

New Features

Enhanced Device Compliance Policies: More precise compliance rules allow for tighter security enforcement, ensuring only authorized devices access corporate resources. These policies help mitigate security risks and improve organizational compliance.

Advanced Reporting Dashboards: The latest reporting dashboards provide IT administrators with an enhanced overview of device health and compliance trends. These dashboards offer data visualization tools to help identify security vulnerabilities and enforce policies more effectively.

Bug Fixes

Resolved Policy Deployment Delays: A recent issue causing delays in policy deployment to devices has been resolved. Policies now apply in a timely manner, ensuring smoother IT operations and security compliance.

Fixed Application Installation Errors: Some Android devices faced errors while installing corporate apps. This fix ensures that all applications install properly without requiring manual intervention, improving the user experience and reducing IT support requests.

Known Issues

iOS Profile Installation Prompts: Some users encounter repeated prompts when installing configuration profiles on iOS devices. Microsoft is currently investigating the cause and working on a fix to improve the installation process.

Windows Autopilot Enrollment Hangs: Certain Windows devices may hang during Autopilot enrollment. A temporary workaround is to restart the device and retry enrollment. Microsoft is actively developing a permanent fix.

Additional Notes

Upcoming Deprecation: Support for Windows 8.1 devices will officially end on July 1, 2025. Organizations should upgrade their systems to a newer Windows version to maintain security, performance, and compatibility with Microsoft services.

Feedback Opportunity: Microsoft is gathering feedback from IT administrators on the new reporting dashboards. Users are encouraged to participate in surveys and provide insights that will help shape future updates.

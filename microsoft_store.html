Microsoft Store App Deployment

1. Assigning Microsoft Store Apps via Intune

Deploy Microsoft Store apps to users and devices via Intune.

Steps:

  • Go to Intune Admin Center.
  • Navigate to Apps → Windows and select + Add.
  • Choose Microsoft Store app (new).
  • Select the app from the Microsoft Store.
  • Configure the app assignment and deployment settings.

2. Monitoring Deployment

Track deployment status in Intune.

Steps:

  • Go to Intune Admin Center → Apps → Monitor.
  • Check installation status and troubleshoot failed deployments.
  • Ensure users have permissions and network access.
